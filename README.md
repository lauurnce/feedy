# feedy

Developer blog feed aggregator with AI-powered summaries.

Tracks updates from Telegram, X, TikTok, Meta, Anthropic, OpenAI and more — fetches, stores, and summarizes them into a daily digest.

## Install

```bash
pip install -e ".[dev]"
```

## Usage

```bash
feedy fetch       # pull latest entries from all sources
feedy list        # show saved entries
feedy digest      # generate today's AI summary
```
